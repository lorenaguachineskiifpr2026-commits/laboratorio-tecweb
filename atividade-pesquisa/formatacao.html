FORMATAÇÃO DO HTML

FORMATAÇÃO DO HTML:

A formatação do HTML é a maneira de organizar e estruturar o conteúdo de uma página usando a linguagem HTMLEm vez de formatar como num editor de texto tipo mudar fonte ou cor diretamente, o HTML usa tags etiquetas para indicar o papel de cada parte do conteúdo.

ELEMENTOS BÁSICOS DA FORMATAÇÃO

  • h1- até h6- Títulos (do maior ao menor)
  • p- Parágrafo
  • br- Quebra de linha
  • strong- Texto importante negrito
  • em- Ênfase itálico
  • ul- / ol- Listas não ordenada / ordenada
  • li- Item da lista
